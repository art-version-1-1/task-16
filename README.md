# Task 16 - Student Search Filter

A small static web app that displays student cards and allows searching through the list by name, marks, class, or address.

## Features

- Live search filtering as you type
- Search button support for manual filtering
- Displays a list of student cards with name, marks, class, and address
- Handles no-results cases gracefully

## Files

- `index.html` - Page structure and search UI
- `style.css` - Styles for layout and student cards
- `script.js` - Data array, search filtering, and DOM rendering logic

## Usage

1. Open `index.html` in a browser.
2. Type any keyword into the search box.
3. Student cards are filtered automatically.
4. Use the Search button to trigger the same filter manually.

## Search behavior

Search matches across:

- Student name
- Marks
- Class
- Address

Search is case-insensitive and accepts partial matches.
